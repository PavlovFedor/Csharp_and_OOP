# Проекты по C# с ООП

# 1) databases-oop
Данная папка содержит выполненные лабораторные работы в рамках курсов, связанных с объектно-ориентированным программированием и базами данных. Основная часть реализована на C# (Windows Forms) и демонстрирует:

- Проектирование абстрактных классов и наследование (подключение к MySQL и PostgreSQL, выполнение запросов, автоматическое закрытие соединения при простое).
- Реализацию классов для выполнения параметризованных запросов к БД (select all, delete, update и др.).
- Создание системы тестирования с различными типами вопросов (одиночный выбор, множественный выбор, упорядочивание) и подсчётом баллов.
- Разработку простого WinForms-приложения – секундомера с записью интервалов (работа с таймером, списками, LINQ).

Все работы содержат обработку исключений и демонстрируют базовые принципы работы с базами данных и ООП на платформе .NET.

# 2) rational-complex-vector-solver-oop
Данная папка содержит выполненные лабораторные работы в рамках курса **«Современные технологии объектно-ориентированного программирования»** (СТООП). Все задания реализованы на языке C# с акцентом на создание математических абстракций, шаблоны (generics), перегрузку операторов и обработку исключений.

## Что реализовано
- **Рациональные числа** – класс с полями «числитель» (целое) и «знаменатель» (натуральное). Поддерживается сокращение дробей, парсинг строк вида `4/5` и `4:5`, арифметические операции (+, -, *, /), сравнение, а также генерация случайного числа на интервале.
- **Комплексные числа** – класс с вещественной и мнимой частями. Реализован парсинг строк `2+4i`, `-4i`, арифметика, равенство, обратный, единичный и нулевой элементы.
- **Интерфейс поля (IField)** – абстракция, определяющая операции сложения, вычитания, умножения, деления, сравнения, парсинга и генерации чисел. Классы рациональных и комплексных чисел реализуют этот интерфейс с использованием универсальных шаблонов (generics), что позволяет создавать обобщённые алгоритмы.
- **Вектор (Vector<T>)** – шаблонный класс, компоненты которого являются элементами поля (рациональные или комплексные числа). Реализованы операторы сложения, вычитания, скалярного умножения, векторное произведение (для размерности 3), умножение на константу, сравнение, парсинг строки вида `(a1, a2, …, an)`, генерация случайного вектора.
- **Система линейных алгебраических уравнений (СЛАУ)** – класс, решающий систему `Ax = b` любым методом (например, методом Гаусса). Реализована проверка существования и единственности решения.

## Ключевые возможности
- Перегрузка операторов (+, -, *, /, ==, !=) для интуитивной работы с числами и векторами.
- Обработка исключений: деление на ноль, неверный формат строки, несовместные системы и т.п.
- Использование обобщений (generics) для работы с разными типами полей (рациональные, комплексные) в едином коде вектора и СЛАУ.
- Парсинг строк и генерация случайных чисел в заданных диапазонах.

## Структура
- `RationalNumber` – класс рациональных чисел
- `ComplexNumber` – класс комплексных чисел
- `IField<T>` – интерфейс поля (с методами сложения, вычитания и т.д.)
- `Vector<T>` – шаблонный класс вектора (T реализует IField<T>)
- `LinearSystem<T>` – класс для решения СЛАУ (использует Vector<T> и матрицу)

# 3) encryptor-decryptor
Простое консольное приложение на C# для симметричного шифрования и дешифрования файлов с использованием алгоритма **TripleDES** (3DES). Генерация ключа и вектора инициализации (IV) происходит автоматически с сохранением в отдельные файлы.

## Возможности
- **Encryptor** – читает исходный файл (`source.txt`), генерирует ключ (`key.dat`) и IV (`iv.dat`), шифрует данные и сохраняет результат в `encrypted.txt`.
- **Decryptor** – читает зашифрованный файл (`encrypted.txt`) и с помощью ранее сохранённых ключа и IV восстанавливает исходный текст в `decrypted.txt`.
