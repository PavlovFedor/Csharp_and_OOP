# Проекты по C# с ООП

# 1) databases-oop
Данная папка содержит выполненные лабораторные работы в рамках курсов, связанных с объектно-ориентированным программированием и базами данных. Основная часть реализована на C# (Windows Forms) и демонстрирует:

- Проектирование абстрактных классов и наследование (подключение к MySQL и PostgreSQL, выполнение запросов, автоматическое закрытие соединения при простое).
- Реализацию классов для выполнения параметризованных запросов к БД (select all, delete, update и др.).
- Создание системы тестирования с различными типами вопросов (одиночный выбор, множественный выбор, упорядочивание) и подсчётом баллов.
- Разработку простого WinForms-приложения – секундомера с записью интервалов (работа с таймером, списками, LINQ).

Все работы содержат обработку исключений и демонстрируют базовые принципы работы с базами данных и ООП на платформе .NET.
